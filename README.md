# library
Api python/Django Rest of books

## 1
```
$  git clone https://github.com/jeffersonsantos26/library.git
```

## 2
```
$  cd library
```

## 3
```
$  python3 -m venv venv
```

## 4
```
$  source venv/bin/activate
```

## 5
```
$  pip3 install django djangorestframework
```

## 6
```
$  python3 manage.py runserver
```
